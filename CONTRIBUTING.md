# Contributing

Thanks for your interest in improving this project!

## How to contribute
1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Commit with clear messages and open a Pull Request against `main`.
4. Add tests or notebook demonstrations where appropriate.
5. Link any sample data or screenshots that help reviewers.

## PR checklist
- [ ] All new code or notebook cells are documented.
- [ ] Added/updated tests or validation steps.
- [ ] Confirmed no sensitive data is included in the commit.
