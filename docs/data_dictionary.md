# Data dictionary

This file documents the main fields used in analysis.

- `client_id` : Unique identifier for customer (anonymized)
- `gender` : 'Male' / 'Female' / 'Other'
- `nationality` : Customer nationality
- `income_band` : Categorical income band (Low / Mid / High)
- `banking_relationship` : Relationship type (Private Bank / Retail / Corporate)
- `engagement_timeframe` : Bucketed account engagement (<5y, 5-10y, 10-20y, >20y)
- `total_loan`, `bank_loan`, `business_lending`, `credit_card_balance`
- `total_deposit`, `bank_deposit`, `savings_amount`, `checking_amount`, `foreign_currency_amount`
- `fees` : Total fees charged
- `record_date` : Date of the record / snapshot
